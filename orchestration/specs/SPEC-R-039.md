# SPEC-R-039: 설정 UI (그래픽·오디오·조작 탭)

**관련 태스크:** R-039

---

## 개요
`SettingsManager`(R-038)와 연동되는 탭 형식 설정 UI 구현.

## 상세 설명
메인 메뉴와 인게임 일시정지 메뉴에서 접근 가능한 설정 화면을 구현한다. 그래픽, 오디오, 조작 3개 탭으로 구성하며, 변경사항은 "적용" 버튼으로 일괄 적용하거나 "취소"로 되돌릴 수 있다. 해상도 변경 시 15초 카운트다운 확인 팝업을 표시하여 화면이 안 보이는 상태에서 잠기는 것을 방지한다.

## 데이터 구조
```csharp
public class SettingsUI : MonoBehaviour
{
    [SerializeField] private GameObject graphicsTab;
    [SerializeField] private GameObject audioTab;
    [SerializeField] private GameObject controlsTab;

    private GameSettings pendingSettings; // 적용 전 임시 설정

    public void Open();
    public void SwitchTab(int tabIndex);
    public void Apply();
    public void Cancel();
    public void ResetDefaults();
}
```

## 연동 경로
| From | To | 방식 |
|------|----|------|
| MainMenuController | SettingsUI | "설정" 버튼 → `SettingsUI.Open()` |
| HUD (Pause Menu) | SettingsUI | ESC → 일시정지 → "설정" 버튼 |
| SettingsUI | SettingsManager | `Apply()` → `SettingsManager.Apply*()` + `Save()` |

## 호출 진입점
- **어디서:** `MainMenuController` → "설정" 버튼, `HUD` → ESC → "설정" 버튼
- **어떻게:** `SettingsUI.Open()` 호출 → 현재 설정값으로 UI 초기화

## 수용 기준
- [ ] 3개 탭 간 전환이 정상 동작
- [ ] 드롭다운에 현재 모니터 지원 해상도 목록이 동적으로 표시됨
- [ ] "적용" 클릭 시 `SettingsManager`에 반영 + 저장
- [ ] "취소" 클릭 시 변경 전 상태로 되돌림
- [ ] 해상도 변경 시 15초 확인 카운트다운 팝업 표시, 미확인 시 이전 해상도로 복원
- [ ] 키 바인딩 재설정: 해당 슬롯 클릭 → "키를 누르세요" → 키 감지 → 중복 검사
- [ ] 메인 메뉴와 인게임 일시정지 메뉴 양쪽에서 접근 가능
