# SPEC-R-041: 출시 전 통합 QA 체크리스트

**관련 태스크:** R-041

---

## 개요
Steam 출시 전 최종 통합 검증 및 성능 최적화 수행.

## 상세 설명
Steam 출시 전에 반드시 확인해야 하는 항목들을 체계적으로 검증한다. Steam API 연동(업적, 클라우드, 오버레이), 오프라인 모드 폴백, 빌드 무결성, 성능 프로파일링, 메모리 누수 체크를 포함한다. Valve의 Steam 출시 체크리스트를 기반으로 프로젝트에 맞게 조정한다.

## 데이터 구조
N/A (테스트 및 검증 태스크)

## 연동 경로
| From | To | 방식 |
|------|----|------|
| QA | SteamManager (R-035) | 초기화 성공/실패 시나리오 검증 |
| QA | SteamCloudStorage (R-036) | 클라우드 저장/불러오기/충돌 검증 |
| QA | SteamAchievementManager (R-037) | 업적 해제/중복/리셋 검증 |
| QA | SettingsManager (R-038) | 모든 설정 조합 적용 검증 |
| QA | BuildScript (R-040) | 릴리즈 빌드 무결성 검증 |

## 호출 진입점
- **어디서:** 모든 R-035~R-040 태스크 완료 후
- **어떻게:** 체크리스트 기반 수동 검증 + 자동 테스트 실행

## 수용 기준
- [ ] Steam 클라이언트 실행 중 게임 시작 → 오버레이(Shift+Tab) 정상 동작
- [ ] Steam 클라이언트 미실행 시 게임 시작 → 오프라인 모드 정상 동작, 크래시 없음
- [ ] 업적 5개 이상 해제/리셋 테스트 완료
- [ ] 클라우드 저장: PC A에서 저장 → PC B에서 불러오기 정상
- [ ] 클라우드 충돌 해소 UI 정상 동작
- [ ] 모든 해상도/전체화면 모드 전환 시 크래시 없음
- [ ] 최소 사양 PC에서 30fps 이상 유지 (Unity Profiler 기준)
- [ ] 1시간 연속 플레이 시 메모리 증가 100MB 미만 (메모리 누수 없음)
- [ ] 릴리즈 빌드에서 콘솔 로그/디버그 UI 비노출
- [ ] 빌드 용량 확인 및 불필요 에셋 제거 완료
- [ ] Steamworks 파트너 사이트에서 스토어 페이지 필수 항목 모두 충족
- [ ] EULA / 개인정보처리방침 페이지 준비 완료
