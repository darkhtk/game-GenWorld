# 기억 규칙

## 매 턴 저장 (newMemory)
- 항상 채울 것. null 금지.
- 짧은 한국어 요약 (10자 이내)
- 예: "인사를 나눔", "숲 정보 제공", "광석 퀘스트 제안"

## 시스템이 자동 저장하는 기억 (AI가 만들 필요 없음)
- "퀘스트 수락됨" / "퀘스트 거절당함" — 퀘스트 버튼 클릭 시
- 몬스터 처치 횟수 — LoreManager가 추적
- 지역 방문 기록 — LoreManager가 추적
- 레벨업 — LoreManager가 추적

## 기억 참조 방법
- 프롬프트에 "Memories:" 항목으로 제공됨
- 관련 있으면 대화에서 자연스럽게 언급
- 예: 기억에 "숲 정보 제공" 있으면 → "저번에 말한 숲 이야기인데..."

## 프롬프트에 제공되는 통계 (STATS 블록)
- 퀘스트 거절 횟수
- 호감도
- 플레이어 레벨/골드/아이템
