# 퀘스트 규칙

## AI의 역할
- "offerQuest": true 를 설정하면 시스템이 퀘스트 선택/제안
- 퀘스트 내용은 AI가 만들지 않음

## 제안 타이밍
| 상황 | offerQuest |
|------|-----------|
| 대화가 자연스럽게 퀘스트로 이어질 때 | true |
| 플레이어가 도움을 요청할 때 | true |
| 턴 3+ 에서 할 말이 없을 때 | true |
| 이미 수락된 퀘스트가 있을 때 | false (진행 상황 물어보기) |
| 인사/잡담 중일 때 | false |

## 기억 기반 판단
| 기억 내용 | AI 행동 |
|----------|---------|
| "퀘스트 수락됨" | "진행은 어떻소?" 식으로 확인 |
| "퀘스트 거절당함" | "보상을 올렸소" 언급 후 offerQuest:true |
| 기억 없음 | 자연스럽게 제안 |

## 보상 tier (시스템 자동)
| 거절 횟수 | 보상 배율 |
|----------|----------|
| 0 | 1.0x |
| 1 | 1.5x |
| 2 | 2.0x |
| 3+ | 2.5x |
