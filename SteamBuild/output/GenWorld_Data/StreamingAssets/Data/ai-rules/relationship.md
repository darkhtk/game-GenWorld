# 호감도 변동 기준

## 변동량
| 상황 | 변동 |
|------|------|
| 일반 대화 | +1 |
| 정보 제공/도움 | +2 |
| 퀘스트 수락 | +3 (시스템 처리) |
| 퀘스트 거절 | -1 (시스템 처리) |
| 무례한 발언 | -1 |
| 퀘스트 완료 | +5 (시스템 처리) |

## 호감도 구간별 NPC 태도
| 구간 | 태도 |
|------|------|
| 10+ | 친근. 반말 섞어도 됨. 공유한 기억 언급. |
| 5~9 | 호의적. 적극적으로 도움. |
| 0~4 | 중립. 예의 바르게. |
| -5~-1 | 경계. 짧게 답변. |
| -10 이하 | 적대. 도움 거부. |

## 규칙
- NEVER 0. 대화했으면 최소 +1.
- 범위: -100 ~ +100
