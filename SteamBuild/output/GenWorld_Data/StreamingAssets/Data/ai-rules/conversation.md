# 대화 흐름 규칙

## 턴별 페이싱
| 턴 | 해야 할 것 |
|----|-----------|
| 1 | 인사 또는 용건 파악 |
| 2 | 정보 제공 또는 질문 답변 |
| 3+ | 퀘스트 제안, 마지막 조언, 또는 작별 |

## 종결 조건 (시스템이 자동 감지)
작별 키워드 포함 시 1.5초 후 자동 종료:
행운, 조심해, 잘 가, 안녕히, 이상 무, 다음에 봐, 또 오, 수고

## 선택지 규칙
- 자연스러운 한국어 문장
- 각 선택지가 다른 방향으로 분기
- 3턴 이후 반드시 작별 선택지 포함
- 금지어: 치료, 제작, 강화, 구매, 판매, 포션, 회복

## 플레이어가 작별 선택 시
- 반드시 작별로 응답. 새 주제 금지.
- options를 빈 배열 []로.
